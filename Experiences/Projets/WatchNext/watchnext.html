🎬 Projet personnel – WatchNext

WatchNext est un site web que j’ai développé pendant mon stage pour mon plaisir personnel, afin de créer une plateforme interactive autour des films.

🌐 Voir le site

🎯 Objectif du projet

L’objectif principal était de permettre aux utilisateurs de parcourir et consulter des films avec toutes leurs informations (titre, synopsis, durée, etc.), voir le trailer de chaque film et profiter d’une interface fluide et interactive.

👥 Utilisateurs et fonctionnalités

  • Tout le monde : peut consulter les films, leurs informations détaillées et regarder les trailers.
  • Administrateurs (mon ami et moi) :
    • Noter les films
    • Ajouter des films dans la section « À voir » pour planifier des séances
    • Ajouter des films dans la section « Déjà regardés »
  • Système de notation automatique qui calcule la moyenne des notes et génère un Top 5 des films les mieux notés.

⚙️ Technologies et compétences utilisées

  • Front-end : HTML, CSS, JavaScript
  • Gestion des films et des données utilisateurs
  • Interaction dynamique avec système de notes
  • Organisation et bonnes pratiques : structuration du projet, gestion du code sur GitHub et utilisation de Visual Studio Code
