Veille technologique

La veille technologique consiste à suivre les évolutions et innovations dans le domaine informatique. Dans le cadre de mon BTS SIO SLAM, j’ai réalisé une veille sur les frameworks frontend.

Sujet de veille : Frontend Frameworks

Je souhaite m’orienter vers le développement web frontend, ce qui explique le choix de ce sujet.

Définition

Un framework frontend est un ensemble d’outils permettant de créer des interfaces web modernes plus rapidement et efficacement.

Frameworks étudiés

React

Développé par Meta. Très utilisé en entreprise, basé sur les composants.

Vue.js

Simple, léger et facile à apprendre. Très apprécié des débutants.

Angular

Développé par Google. Framework complet mais complexe.
